# Andrew's nvim dotfiles
These are my dotfiles for neovim

# Installation
### Wakatime
1. Put wakapi.dev API key into the .env file after you copy to .env.example
2. Install Wakatime CLI
    - I reccomend using `winget install -e --id Wakatime.CLIWakatime`
3. Set Wakatime CLI path variable
